groups The group defines a logical collection of DragDrop objects that are related. Instances only get events when interacting with other DragDrop object in the same group. This lets us define multiple groups using a single DragDrop subclass if we want.  
hasOuterHandles By default, drags can only be initiated if the mousedown occurs in the region the linked element is. This property is set to true if outer handles are defined. @default false
